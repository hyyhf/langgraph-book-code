# 快速开始指南

## 5分钟快速体验

### 第一步: 确保环境配置正确

确保你已经在`code_for_learning/.env`文件中配置了OpenAI API密钥:

```bash
OPENAI_API_KEY=your_api_key_here
OPENAI_BASE_URL=https://api.openai.com/v1
```

### 第二步: 运行基础示例

```bash
cd code_for_learning
uv run python chapter10/example_basic.py
```

4. 保存报告到`research_report.txt`

## 常见问题

### Q1: 如何修改研究主题?

在`example_basic.py`中修改:
```python
topic = "你的研究主题"
```

### Q2: 如何启用人工审核?

在`main.py`中选择示例2,或者:
```python
graph = create_research_agent(with_human_review=True)
```

### Q3: 如何使用真实的搜索API?

修改`tools.py`中的`search_web`函数:
```python
from langchain_tavily import TavilySearch

@tool
def search_web(query: str) -> str:
    """使用Tavily搜索"""
    tavily = TavilySearch(max_results=5)
    results = tavily.invoke(query)
    return results
```

### Q4: 如何保存报告到不同位置?

在`example_basic.py`中修改:
```python
output_file = "outputs/my_report.txt"  # 修改保存路径
```

### Q5: 如何调整LLM参数?

修改`nodes.py`中的`get_llm`函数:
```python
def get_llm(streaming: bool = True):
    return ChatOpenAI(
        model="gpt-4o-mini",
        temperature=0.7,  # 修改温度
        streaming=streaming
    )
```
