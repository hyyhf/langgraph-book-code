# 快速开始指南

## 5分钟快速体验

### 1. 环境检查

确保已安装Python 3.10+和uv:

```bash
python --version
uv --version
```

### 2. 配置API密钥

编辑项目根目录的`.env`文件,确保包含以下配置:

```env
# AI模型配置
OPENAI_BASE_URL=https://api.deepseek.com/v1
OPENAI_API_KEY=your_api_key_here
OPENAI_MODEL_NAME=deepseek-chat

# 高德地图API配置
AMAP_API_KEY=your_amap_key_here
AMAP_BASE_URL=https://restapi.amap.com/v3
```

### 3. 运行基础测试

```bash
cd code_for_learning/chapter12
uv run python test_basic.py
```

期望输出:

```
================================================================================
武汉旅行规划助手 - 基础测试
================================================================================
测试1: 创建智能体... 成功!
测试2: 导入工具... 成功!
测试3: 状态定义... 成功!
测试4: 简单运行测试... 成功!

通过: 4/4
所有测试通过!
```

### 4. 运行自动示例

```bash
uv run python example_auto.py
```

这个示例会:
- 自动创建一个2天的武汉旅行计划
- 调用高德地图API获取信息
- 自动确认计划
- 生成HTML文档

### 5. 运行交互式版本

```bash
uv run python main.py
```

按照提示:
1. 输入您的旅行需求,例如: "我想在武汉玩3天,喜欢历史文化和美食"
2. 等待智能体收集信息并生成计划
3. 审核计划,输入"确认"或提出修改建议
4. 查看生成的HTML文档

### 6. 查看生成的文档

HTML文档保存在`outputs/`目录下,文件名格式为:

```
travel_plan_武汉_20251025_172230.html
```

在浏览器中打开即可查看精美的旅行计划。

## 常见问题

### Q1: API调用失败怎么办?

**A:** 检查以下几点:
1. API密钥是否正确
2. 网络连接是否正常
3. API配额是否充足
4. 高德地图API是否已开通相关服务

### Q2: 如何修改旅行天数?

**A:** 在`main.py`中修改`travel_days`参数:

```python
initial_state = {
    ...
    "travel_days": 5,  # 改为5天
    ...
}
```

### Q3: 如何更换城市?

**A:** 修改`destination`参数和用户请求:

```python
user_request = "我想在北京玩3天..."
initial_state = {
    ...
    "destination": "北京",
    ...
}
```

### Q4: 如何自定义HTML样式?

**A:** 在`nodes.py`的`html_generator_node`函数中修改`html_prompt`,添加您想要的样式要求。

### Q5: 如何查看图结构?

**A:** 运行可视化脚本:

```bash
uv run python visualize_graph.py
```

会生成`wuhan_travel_agent_graph.mmd`文件,可以在支持Mermaid的编辑器中查看。
